# ANEXO 2: FUNCIONALIDADES IMPLEMENTADAS Y PENDIENTES

ESTE DOCUMENTO DETALLA EL ESTADO DE LAS FUNCIONALIDADES REQUERIDAS PARA LA FASE 1 DEL PROYECTO BYTEBAKERY.

## FUNCIONALIDADES IMPLEMENTADAS (100% COMPLETADAS)
- RESPONSIVE DESIGN (TABLETS, CELULARES, DESKTOP)
- PORTADA CON TITULOS, LOGO Y BARRA DE MENU FIJA
- DISENO A TRES COLUMNAS EN LA PAGINA PRINCIPAL
- EFECTO FADE (APARECER SUAVE) USANDO FRAMER MOTION
- MODALES DE BOOTSTRAP PARA LOGIN Y REGISTRO
- COMPONENTE COLLAPSE EN LA PORTADA
- MENU DROPDOWN EN LA BARRA DE NAVEGACION
- PIE DE PAGINA (FOOTER) CON REDES SOCIALES Y EQUIPO
- ROUTER (NEXT.JS APP ROUTER) CON RUTAS PUBLICAS Y PRIVADAS
- PROTECCION DE RUTAS PRIVADAS (REDIRIGE AL INICIO SI NO HAY LOGIN)
- SIMULACION DE LOGIN (ADMIN / 1234) SIN BASE DE DATOS
- CARRUSEL DE IMAGENES EN LA PORTADA
- ACORDEON PARA PREGUNTAS FRECUENTES
- VIDEOS EMBEBIDOS (SECCION ARTE DE LA PASTELERIA)
- SONIDO DE AMBIENTE CON REPRODUCTOR ESTILO PILL
- DROP CAP (PRIMERA LETRA GRANDE Y A COLOR EN TEXTOS)
- LOGO ACTUALIZADO A ByeBakery.png (SOLICITUD DE USUARIO)
- FORMULARIOS USANDO REACT-BOOTSTRAP/FORM (CONTACTO Y PEDIDOS)
- ORGANIZACION DE CARPETAS (COMPONENTS, APP, PUBLIC, STYLES)
- LUPA PARA VER CONTENIDO DE PASSWORD (ICONO OJO)
- CAPTCHA DE SEGURIDAD (OPERACIONES MATEMATICAS) EN LOGIN

- PASARELA DE PAGOS REAL (SOLO SIMULACION DE PEDIDO)
