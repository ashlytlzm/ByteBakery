# ANEXO 1: PRINCIPIOS DE DISENO EMPLEADOS

EN ESTE PROYECTO SE HAN APLICADO LOS SIGUIENTES PRINCIPIOS DE DISENO WEB PARA LOGRAR UNA EXPERIENCIA PREMIUM Y FUNCIONAL:

1. CONTRASTE DE COLOR: SE UTILIZO UNA PALETA BASADA EN TONOS CREMA (VAINILLA) Y VINO OSCURO (REPOSTERIA CLASICA) PARA GARANTIZAR QUE EL TEXTO SEA LEGIBLE Y LOS ELEMENTOS IMPORTANTES RESALTEN.
2. RESPONSIVE DESIGN: EL SITIO SE ADAPTA AUTOMATICAMENTE A CELULARES, TABLETS Y COMPUTADORES DE ESCRITORIO USANDO BOOTSTRAP Y TAILWIND CSS.
3. JERARQUIA VISUAL: LOS TITULOS USAN UNA TIPOGRAFIA SERIF (CHANGO) MAS GRANDE PARA GUIAR LA MIRADA DEL USUARIO.
4. CONSISTENCIA: LA BARRA DE NAVEGACION (HEADER) Y EL PIE DE PAGINA (FOOTER) SON FIJOS Y SE MANTIENEN IGUAL EN TODAS LAS PAGINAS.
5. ESPACIADO (WHITE SPACE): SE DEJO ESPACIO SUFICIENTE ENTRE ELEMENTOS PARA QUE LA INTERFAZ NO SE VEA SOBRECARGADA, SIGUIENDO EL ESTILO DE CRUMBL COOKIES.
6. TIPOGRAFIA: USO DE FUENTES DE GOOGLE FONTS (PLAYFAIR DISPLAY Y LATO) PARA COMBINAR ELEGANCIA Y LEGIBILIDAD.
7. ACCESIBILIDAD: CONTRASTE ADECUADO ENTRE FONDO Y TEXTO (AA/AAA).
8. ANIMACIONES SUTILES: USO DE FRAMER MOTION PARA EFECTOS DE FADE Y DESPLAZAMIENTO QUE MEJORAN LA SENSACION DE CALIDAD.
